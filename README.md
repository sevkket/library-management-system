# 📚 Kütüphane Otomasyon Sistemi

Bu proje, C# Windows Forms kullanılarak geliştirilmiş basit bir kütüphane otomasyon sistemidir.

## 🚀 Özellikler

- 🔐 Giriş sistemi (Admin / Kullanıcı)
- 📖 Kitap listeleme
- ➕ Kitap ekleme (Admin)
- ✏️ Kitap güncelleme (Admin)
- ❌ Kitap silme (Admin)
- 📥 Kitap ödünç alma (Kullanıcı)

## 👤 Kullanıcı Bilgileri

- Admin → kullanıcı adı: `admin` | şifre: `1234`
- Kullanıcı → kullanıcı adı: `user` | şifre: `1234`

## 🛠️ Kullanılan Teknolojiler

- C#
- Windows Forms
